# Match predictor

A few different predictors for English football results.

## Install dependencies

```shell
make install
```

## Backend

### Run tests

```shell
make backend/test
```

### Run an accuracy report

```shell
make backend/run
```

## Frontend

### Run tests

```shell
make frontend/test
```

### Run server

```shell
make frontend/run
```

## Integration tests

```shell
make integration/test
```
